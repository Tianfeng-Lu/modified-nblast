# modified NBLAST
An implementation of NBLAST with graph-constrained rigid transformation

# files
- `rigid_invariant_NBLAST.py`: the main implementation of the NBLAST algorithm after rigid transformation
- `example_code.ipynb`: an example notebook showing how to use functions in `rigid_invariant_NBLAST.py`
- `GroupRigidRegistration.py`: a modified version of the cpd algorithm for a translation-free rigid registration with group indicators and graph constraints
- `example_data/`: example neuron skeletons for testing the code
